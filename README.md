AI & Machine Learning Mastery Course
====================================

Course Description
------------------

Welcome to the AI & Machine Learning Mastery Course! This program is designed to help beginners learn core Artificial Intelligence and Machine Learning concepts while also practicing proper Python coding habits. You will read short lessons, study guided code examples, type and run the examples yourself, and use quizzes to check your understanding.
Course Structure
----------------

This course is divided into six progressive modules, each building upon the last.

1. **AI/ML Foundations:** Understand the core concepts of AI and Machine Learning.

2. **Core Machine Learning:** Master data preprocessing and fundamental ML algorithms.

3. **Deep Learning:** Dive into neural networks and learn to build deep learning models.

4. **Natural Language Processing:** Learn how to enable computers to understand human language.

5. **Computer Vision:** Explore how to enable machines to "see" and interpret images.

6. **Advanced Topics & Capstone:** Apply your knowledge to a comprehensive, end-to-end project.

Prerequisites
-------------

* No AI or Machine Learning experience is required.

* Basic Python familiarity helps, but the hands-on examples include comments and explanations so beginners can learn what each line is doing.

How to Get Started
------------------

1. **Clone the Repository:** Clone this repository to your local machine using `git clone [repository-url]`.

2. **Set up the Environment:** The course uses several Python libraries. It's highly recommended to use a virtual environment to avoid conflicts.
   
   * Navigate to the project directory in your terminal.
   
   * Create and activate a virtual environment:
     
     * **macOS / Linux:**
          python3 -m venv venv
          source venv/bin/activate

    * **Windows:**
    
          python -m venv venv
          .\venv\Scripts\activate

* Install all necessary dependencies with a single command:
  
      pip install -r requirements.txt
3. **Start Learning:** Serve the project locally, then open it in your browser:

      python -m http.server 8000

   Then visit `http://localhost:8000/`.

How to Use the Course
---------------------

* Read each module before jumping into the code.
* Type the code examples into a notebook or Python file instead of only copying and pasting.
* Read the "What you are about to code" section before running each example.
* Make one small change at a time, run the code again, and observe what changed.
* Complete each knowledge check before moving to the next module.

The final module includes a safe AI agent planning extension. Keep agent projects narrow, legal, privacy-respecting, and human-approved.

Enjoy the course!
