This final module introduces several advanced and modern topics in AI and Machine Learning. Reinforcement Learning (RL) is a type of machine learning where an agent learns to make decisions by performing actions in an environment to maximize a cumulative reward.

Capstone Project: Financial News Sentiment Dashboard

The Capstone Project is the final, hands-on component where you apply the knowledge and skills from all six modules to a single, comprehensive problem. You'll build an end-to-end Financial News Sentiment Dashboard, covering data acquisition, NLP, model training, and deployment.

Final Extension: Build Your Own Safe AI Agent

After completing the core capstone, you can design your own beginner AI agent. An AI agent is a program that receives input, uses rules or a model to reason about that input, and recommends or performs a bounded action.

The goal is to build something useful and safe: a study helper, task organizer, budgeting assistant, writing coach, customer-support triage tool, or research summarizer. Your agent should help a person make better decisions without hiding what it is doing.

Safe Agent Rules

  • Keep the scope narrow: define exactly what the agent is allowed to help with.
  • Keep a human in control: the agent can suggest actions, but a person should approve important decisions.
  • Protect privacy: do not collect passwords, private keys, banking logins, medical records, or unnecessary personal data.
  • Stay legal and non-harmful: do not build agents for hacking, surveillance, fraud, harassment, weaponization, or bypassing security systems.
  • Explain the output: the agent should show why it recommended something, not just give an answer.
