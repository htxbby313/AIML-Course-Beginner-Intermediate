Introduction to Neural Networks

Deep Learning is a powerful subset of Machine Learning that uses artificial neural networks with multiple layers to learn complex patterns. A neural network consists of layers of neurons, with non-linear activation functions like ReLU and Softmax applied to their outputs.

Coding Basics: Functions and Step-by-Step Pipelines

Neural network code looks advanced, but it is still built from beginner coding ideas. You import tools, create data, define a model, configure how it learns, then run training.

Think of the model as a pipeline. Each layer receives numbers, transforms them, and passes the result forward. The code matters because every layer you add changes what patterns the model can learn.

Hands-on: Building a Simple Neural Network

Let's build a simple neural network using Keras on top of TensorFlow to see how it works in practice.

What you are about to code

  • NumPy arrays store the training examples and labels.
  • Weights are the numbers a simple model uses to make a prediction.
  • sigmoid turns any score into a value between 0 and 1.
  • errors show how far the prediction is from the correct answer.
  • training repeatedly adjusts the weights to reduce mistakes.
  • predictions convert probabilities into 0 or 1 class labels.

Module 3: Neural Networks Notebook

import numpy as np

# Create a tiny dataset for an AND-style classification problem.
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([0, 0, 0, 1])

# Start with simple weights and a bias.
weights = np.array([0.0, 0.0])
bias = 0.0
learning_rate = 0.5

# The sigmoid function turns a raw score into a probability.
def sigmoid(score):
    return 1 / (1 + np.exp(-score))

# Train for several passes through the tiny dataset.
for epoch in range(80):
    raw_scores = X @ weights + bias
    probabilities = sigmoid(raw_scores)
    errors = probabilities - y

    # Update weights and bias using the average error.
    weights = weights - learning_rate * (X.T @ errors) / len(X)
    bias = bias - learning_rate * errors.mean()

# Convert probabilities into class predictions.
predictions = (sigmoid(X @ weights + bias) >= 0.5).astype(int)

print("Learned weights:", weights)
print("Learned bias:", round(float(bias), 3))
print("Predictions:", predictions.tolist())
