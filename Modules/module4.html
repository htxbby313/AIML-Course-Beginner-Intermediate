Text Preprocessing

Natural Language Processing (NLP) is a subfield of AI that focuses on enabling computers to understand human language. Text Preprocessing is a crucial step before any NLP task.

Key steps include: Tokenization, Lowercasing, Stop Word Removal, Stemming (a crude word-root reduction), and Lemmatization (a more sophisticated word-root reduction considering context).

What you are about to code

  • lower() normalizes text so "Natural" and "natural" are treated the same.
  • word_tokenize splits a sentence into individual tokens.
  • stopwords provides common words that are often removed before modeling.
  • word.isalpha() keeps word tokens and removes punctuation tokens.
  • The final filtered list is a cleaner version of the original sentence.

Hands-on: NLP Basics

Before running this code, you may need to download NLTK data. Open a Python terminal and run nltk.download('punkt') and nltk.download('stopwords').

import nltk
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords
from nltk.stem import PorterStemmer

sample_text = "Natural Language Processing (NLP) is a fascinating field."

# Step 1: Tokenization and Lowercasing.
# This breaks the text into individual words or tokens and converts them to lowercase.
text_lower = sample_text.lower()
tokens = word_tokenize(text_lower)
print(f"Tokens: {tokens}")

# Step 2: Removing Stop Words.
# Stop words are common words (like 'is', 'a', 'the') that are often
# filtered out because they add little semantic value.
stop_words = set(stopwords.words('english'))
filtered_tokens = [word for word in tokens if word.isalpha() and word not in stop_words]
print(f"Filtered tokens: {filtered_tokens}")
