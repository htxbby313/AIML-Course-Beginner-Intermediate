Text Preprocessing

Natural Language Processing (NLP) is a subfield of AI that focuses on enabling computers to understand human language. Text Preprocessing is a crucial step before any NLP task.

Key steps include: Tokenization, Lowercasing, Stop Word Removal, Stemming (a crude word-root reduction), and Lemmatization (a more sophisticated word-root reduction considering context).

Coding Basics: Strings, Lists, and Filtering

Text starts as a string, which is how code stores words and sentences. NLP usually turns that string into a list of smaller pieces called tokens.

Filtering is a core coding skill: keep what matters and remove what does not. In NLP, that means removing punctuation or common words so the model can focus on meaningful language.

What you are about to code

  • lower() normalizes text so "Natural" and "natural" are treated the same.
  • replace(...) removes punctuation for this beginner example.
  • split() breaks a sentence into individual tokens.
  • stop_words stores common words that are often removed before modeling.
  • word not in stop_words keeps the words that carry more meaning.
  • The final filtered list is a cleaner version of the original sentence.

Hands-on: NLP Basics

This version uses built-in Python string tools so it can run directly in the browser course.

sample_text = "Natural Language Processing (NLP) is a fascinating field."

# Step 1: Lowercase the sentence.
# This makes "Natural" and "natural" match the same way.
text_lower = sample_text.lower()

# Step 2: Remove punctuation for this beginner example.
clean_text = text_lower.replace("(", "").replace(")", "").replace(".", "")

# Step 3: Split the sentence into word tokens.
tokens = clean_text.split()
print(f"Tokens: {tokens}")

# Step 4: Remove common stop words.
# Stop words are common words that often add little meaning for a model.
stop_words = {"is", "a", "the", "and", "of", "to"}
filtered_tokens = [word for word in tokens if word not in stop_words]

print(f"Filtered tokens: {filtered_tokens}")
