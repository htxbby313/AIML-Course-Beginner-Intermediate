MODULE 5: COMPUTER VISION

Image Basics

Computer Vision (CV) is a field of AI that enables computers to "see" and interpret digital images and videos. An image is a grid of pixels, represented using color models like RGB with three channels (Red, Green, Blue).

Image Preprocessing

Image Preprocessing includes: Resizing, Normalization (scaling pixel values), and Augmentation (creating new training examples).

Coding Basics: Images Are Numbers

To a computer, an image is not a picture first. It is a grid of numbers. Each pixel has values that describe color and brightness.

This matters for AI because computer vision models learn from those numbers. Code is the bridge between what you see on screen and what the model receives as input.

How to think about the code you will write later

  • Loading an image turns an image file into numbers.
  • Resizing makes every image the same shape so the model can process batches consistently.
  • Normalization usually scales pixel values from 0-255 down to 0-1.
  • Augmentation creates modified copies, such as flipped or rotated images, to help the model generalize.
