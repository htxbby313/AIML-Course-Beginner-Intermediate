Data Preprocessing

This module dives into the essential steps of preparing your data. Data Preprocessing is a crucial step that involves cleaning, organizing, and transforming raw data into a suitable format for ML algorithms.

Key steps include: Handling Missing Values, Data Transformation (Scaling/Encoding), and Feature Engineering.

What you are about to code

  • np.nan represents a missing value in the sample data.
  • median() finds a middle value that can replace the missing score.
  • fillna(...) fills missing values so the model does not receive blanks.
  • get_dummies(...) converts text categories into numeric columns.
  • The final table is cleaner and easier for machine learning algorithms to read.

Hands-on: Data Cleaning

import pandas as pd
import numpy as np

# Sample messy dataset with missing values (np.nan) and categorical data
data = {
    'satisfaction_score': [4, 5, 2, np.nan, 5],
    'product_type': ['A', 'B', 'A', 'C', 'B']
}
df = pd.DataFrame(data)
print("Original DataFrame:\n", df)

# Fill missing numerical values with the median. This is a common
# technique to handle missing data without discarding the entire row.
median_score = df['satisfaction_score'].median()
df['satisfaction_score'] = df['satisfaction_score'].fillna(median_score)

# One-hot encode the 'product_type' categorical column.
# This converts categorical data into a numerical format that ML
# algorithms can easily process.
df = pd.get_dummies(df, columns=['product_type'], prefix='product')
print("\nCleaned and Encoded DataFrame:\n", df)

Checkpoint before the quiz: A model needs clean numeric inputs. This example fixes one missing number and turns product labels into model-readable columns.
